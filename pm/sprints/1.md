- [x] Define the chatbot’s purpose, scope, and business objectives.

  - [x] **User Interface**: Interaction layer through web widgets or messaging platforms.
  - [x] **NLP Engine**: Chosen NLP tool (e.g., Dialogflow, Rasa).
